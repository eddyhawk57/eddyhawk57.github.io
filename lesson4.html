Lesson 4: Building Your First Machine Learning Model

Introduction

Now that you have clean and preprocessed data, it's time to build your first machine learning model! In this lesson, we'll use the popular scikit-learn library in Python to create a simple classifier.

Supervised Learning Overview

We'll start with supervised learning, where the model learns from labeled data. One of the most common algorithms for classification is the Logistic Regression model.

Steps to Build a Model

  • Import libraries and load the data
  • Split the data into training and testing sets
  • Train the model using training data
  • Evaluate the model using testing data

Python Example


from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score
import pandas as pd

# Load your dataset
df = pd.read_csv('data.csv')

# Features and target
X = df[['feature1', 'feature2']]
y = df['label']

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Build and train model
model = LogisticRegression()
model.fit(X_train, y_train)

# Predictions
predictions = model.predict(X_test)

# Accuracy
print("Accuracy:", accuracy_score(y_test, predictions))

Conclusion

You’ve just built and tested your first machine learning model! In the next lesson, we’ll explore how to improve its performance using evaluation metrics and model tuning.
