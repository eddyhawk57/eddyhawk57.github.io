Lesson 22: Building a Text Classifier with Scikit-learn

Introduction

In this lesson, you'll learn how to build a simple text classifier using Scikit-learn. Text classification is widely used for spam detection, sentiment analysis, and topic labeling.

Tools You’ll Use

  • Scikit-learn – A powerful machine learning library
  • TfidfVectorizer – Converts text into numerical format
  • MultinomialNB – A Naive Bayes classifier suitable for text data

Example Code


from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import make_pipeline

# Sample training data
texts = ["I love this product", "This is a bad movie", "Amazing experience", "Terrible service"]
labels = ["positive", "negative", "positive", "negative"]

# Build a pipeline
model = make_pipeline(TfidfVectorizer(), MultinomialNB())

# Train the model
model.fit(texts, labels)

# Make a prediction
print(model.predict(["This is fantastic"]))

Conclusion

You’ve now seen how to build a simple text classifier in Python. This foundational skill is crucial for tasks like spam detection, product review analysis, and more. Next, we’ll explore training deep learning models for text data.
