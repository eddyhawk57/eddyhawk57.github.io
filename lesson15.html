Lesson 15: Hyperparameter Tuning

What Are Hyperparameters?

Hyperparameters are settings used to control the learning process of a machine learning model. They are set before training begins and can significantly affect performance.

Examples of Hyperparameters

  • Learning rate
  • Number of hidden layers or neurons
  • Number of epochs
  • Batch size
  • Regularization parameters

Why Tune Hyperparameters?

Proper tuning helps find the best combination of settings that improve model accuracy and reduce overfitting or underfitting.

Common Tuning Methods

  • Grid Search: Tries all combinations from a grid of parameters
  • Random Search: Tries random combinations for faster results
  • Bayesian Optimization: Uses probability to choose the next best parameters to test

Sample Code (Grid Search)


from sklearn.model_selection import GridSearchCV
from sklearn.ensemble import RandomForestClassifier

param_grid = {
    'n_estimators': [50, 100, 150],
    'max_depth': [None, 10, 20],
}

grid = GridSearchCV(RandomForestClassifier(), param_grid, cv=3)
grid.fit(X_train, y_train)

print("Best Parameters:", grid.best_params_)
