Lesson 18: Convolutional Neural Networks (CNNs)

Introduction to CNNs

Convolutional Neural Networks (CNNs) are specialized deep learning models designed for processing image data. Theyâ€™re built to automatically detect patterns such as edges, shapes, and objects in images.

Key Components of CNNs

  • Convolutional Layer: Applies filters to extract features from the input image.
  • Pooling Layer: Reduces spatial dimensions, making computation efficient (e.g., Max Pooling).
  • Fully Connected Layer: The final classification layer, similar to traditional neural networks.

Use Cases

  • Face recognition
  • Medical image diagnosis
  • Self-driving car vision systems

Example Using Keras


from keras.models import Sequential
from keras.layers import Conv2D, MaxPooling2D, Flatten, Dense

model = Sequential()
model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(64, 64, 3)))
model.add(MaxPooling2D(pool_size=(2, 2)))
model.add(Flatten())
model.add(Dense(128, activation='relu'))
model.add(Dense(1, activation='sigmoid'))

model.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])
model.fit(X_train, y_train, epochs=10, batch_size=32)
