Lesson 3: Data Collection and Preprocessing

Importance of Data in Machine Learning

Data is the foundation of any AI or machine learning model. The quality, quantity, and relevance of your data directly affect the performance of your model. Without good data, even the most powerful algorithms will fail.

Sources of Data

  • Public Datasets: Kaggle, UCI Machine Learning Repository, Google Dataset Search
  • APIs: Twitter API, OpenWeatherMap API, YouTube API
  • Web Scraping: Extracting data from websites using tools like BeautifulSoup or Scrapy

Steps in Data Preprocessing

  • Data Cleaning: Handle missing values, remove duplicates, fix errors
  • Normalization: Scale numerical values to a consistent range
  • Encoding: Convert categorical data into numbers using techniques like One-Hot Encoding
  • Splitting: Divide data into training and testing sets

Example in Python

Here's a simple preprocessing example using pandas:


import pandas as pd
from sklearn.preprocessing import StandardScaler

# Load data
df = pd.read_csv('data.csv')

# Fill missing values
df.fillna(0, inplace=True)

# Normalize features
scaler = StandardScaler()
df[['feature1', 'feature2']] = scaler.fit_transform(df[['feature1', 'feature2']])
