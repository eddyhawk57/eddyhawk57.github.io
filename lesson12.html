Lesson 12: Transfer Learning with Pretrained Models

What is Transfer Learning?

Transfer learning leverages a model trained on a large dataset (like ImageNet) and reuses it for a different, often smaller, dataset. This saves time and computation and often yields better performance.

How It Works

  • Use the base of a pretrained model (e.g., VGG16, ResNet)
  • Freeze the base layers to retain learned features
  • Add your own dense layers for the new classification task

Example: Using VGG16


from tensorflow.keras.applications import VGG16
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Flatten, Dense

# Load VGG16 without the top layer
base_model = VGG16(weights='imagenet', include_top=False, input_shape=(224, 224, 3))
base_model.trainable = False

# Add custom layers
model = Sequential([
    base_model,
    Flatten(),
    Dense(128, activation='relu'),
    Dense(10, activation='softmax')
])

When to Use Transfer Learning

  • When you have limited data
  • For faster development
  • To benefit from large-scale pretrained knowledge
